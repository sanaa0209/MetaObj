Il codice è organizzato nei seguenti moduli:

Classi
def-class:
Definisce una nuova classe specificandone il nome, la lista di classi parent e i campi e metodi associati.

add-class-spec: Aggiunge le specifiche di una classe a una tabella hash.

class-spec-parents: Restituisce la lista di classi parent di una data classe.

class-spec-fields: Restituisce la lista dei campi di una data classe.

class-spec-methods: Restituisce la lista dei metodi di una data classe.

is-class: Verifica se una classe è stata definita.

Modulo Istanze

make: Crea un'istanza di una classe specifica con i valori forniti per i campi.

is-instance: Verifica se un oggetto è un'istanza di una determinata classe.

field: Restituisce il valore di un campo specifico di un'istanza.

fields: Restituisce i valori di uno o più campi specifici di un'istanza.

Metodi
get-methods: Estrae la lista di metodi da una specifica di classe.

process-method: Elabora e memorizza il codice di un metodo.

invoke-method: Invoca un metodo su un'istanza.
